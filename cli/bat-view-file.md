---
Title: View File with Syntax Highlighting
Description: View file contents with syntax highlighting using bat (aliased to cat)
Tags:
  - cli
  - file-viewing
  - bat
---

```bash
# View file with syntax highlighting
cat file.ts

# View multiple files
cat file1.ts file2.ts

# View JSON with highlighting
cat package.json

# View markdown with highlighting
cat README.md
```
