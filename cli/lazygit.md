Title: Launch Lazygit Terminal UI
Description: Open lazygit terminal UI for interactive git operations
Tags:
  - cli

  - tui
---

```bash
# Launch lazygit
lg

# Interactive git interface with:
# - Visual diff view
# - Easy staging/unstaging
# - Commit management
# - Branch switching
# - Interactive rebase
# - Stash management

# Keyboard shortcuts:
# - Tab: Switch panels
# - Enter: Select/drill down
# - Space: Stage/unstage
# - c: Commit
# - p: Push
# - P: Pull
# - q: Quit
```
