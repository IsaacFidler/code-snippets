---
Title: Quick Project Switcher
Description: Fuzzy find and jump to any project in ~/dev-projects
Tags:
  - cli
  - fzf
  - navigation
  - projects
---

```bash
# Jump to any project instantly
fzf-projects

# 1. Shows all top-level projects in ~/dev-projects
# 2. Type project name (e.g., "farmer" or "code-snippets")
# 3. See tree preview of project
# 4. Press Enter to cd into project

# Use case: "I want to quickly switch to another project"
# Alternative: Use 'z project-name' after you've visited it once
```
