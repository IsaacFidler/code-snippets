---
Title: Global File Search in Projects
Description: Search for any file across all your dev projects with preview
Tags:
  - cli
  - fzf
  - file-search
  - global
---

```bash
# Search ALL files in ~/dev-projects
fzf-global-file

# 1. Searches across ALL your projects
# 2. Type to fuzzy search (e.g., "Button.tsx")
# 3. See syntax-highlighted preview
# 4. Press Enter to print path (copy/use it)

# Use case: "I need to find that config file from another project"
```
