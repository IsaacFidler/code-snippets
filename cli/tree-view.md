---
Title: Tree View Directory Structure
Description: View directory structure as a tree with icons and git status
Tags:
  - cli
  - navigation
  - eza
  - tree
---

```bash
# View directory tree with icons
tree

# Limit depth to 2 levels
tree -L 2

# Limit depth to 3 levels
tree -L 3

# Include hidden files
tree -a

# Limit depth and include hidden
tree -L 2 -a
```
