---
Title: Quick Parent Directory Navigation
Description: Navigate up directory levels quickly without typing full paths
Tags:
  - cli
  - navigation
  - shortcuts
---

```bash
# Go up one directory (parent)
..

# Go up two directories (grandparent)
...

# Go up three directories (great-grandparent)
....

# Examples:
# From: /Users/isaacfidler/dev-projects/farmer-agent-dir/src/components
.. # → /Users/isaacfidler/dev-projects/farmer-agent-dir/src
... # → /Users/isaacfidler/dev-projects/farmer-agent-dir
.... # → /Users/isaacfidler/dev-projects
```
