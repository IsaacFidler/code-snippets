Title: Fuzzy Find and Change Directory
Description: Fuzzy search for a directory in current location and cd into it
Tags:
  - cli

---

```bash
# Fuzzy find directory and cd into it (from current location)
fzf-cd

# Also works with Alt+C keyboard shortcut
# (Option+C on Mac - requires Ghostty config)

# 1. Shows all subdirectories
# 2. Type to fuzzy search
# 3. See tree preview
# 4. Press Enter to cd

# Use fzf-projects for global project search
```
