---
Title: Global Directory Search in Projects
Description: Search for any directory across all your dev projects with tree preview
Tags:
  - cli
  - fzf
  - directory-search
  - global
  - navigation
---

```bash
# Search ALL directories in ~/dev-projects
fzf-global-dir

# 1. Searches across ALL your projects
# 2. Type to fuzzy search (e.g., "components")
# 3. See tree preview of directory contents
# 4. Press Enter to print path

# Use case: "I need to find the utils folder from another project"
```
