---
Title: Fuzzy Find and Edit File
Description: Fuzzy search for a file in current directory and open in Zed
Tags:
  - cli
  - fzf
  - editor
  - zed
---

```bash
# Fuzzy find file and open in Zed
fzf-edit

# 1. Shows all files in current directory (recursively)
# 2. Type to fuzzy search
# 3. Use arrow keys to navigate
# 4. Press Enter to open in Zed

# Great for: "I need to edit a file but don't remember exact path"
```
